# Live-Line FrontEnd
A real-time MTA train navigation app that uses the user location to show them the closest train stations and the times each train arrives at said station.

## Pages
- Profile
   - Displays user's info and allows theme toggle
   - Allows user to edit their info as the please
- Login/Sign Up
   - Allows users to log into their accounts in order to access their saved favorites 
- Help
   - Displays info about the site and FAQs to help users navigate the app
- Favorites
   - Displays all user's saved locations and pinned train lines
